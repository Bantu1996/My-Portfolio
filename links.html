in having to earn income and have more time to themselves.

with Handlebars for templating, and a MySQL database.

Click here to see GitHub Repository

Dancing Zoo App helps young children in having to learn at a young age.
Through the use of this app children will improve their memory and have
fun when imitating the moves they will be seeing on the video.
This app was built using express-flavoured NodeJS,
with API file for calling the API and rendering the
results JSON data to the screen usinng HandlebarsJS,
and a MySQL database. It was deployed to Heroku and
GitHub using Git on the terminal. This was a team project for the Codex Hackathon; my contributions were the front end design, back end designs and creating the API .
